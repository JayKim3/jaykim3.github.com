> 단어정렬

- 알파벳 소문자로 이루어진 N개의 단어가 들어오면 아래와 같은 조건에 따라 정렬하는 프로그램을 작성하시오.

1. 길이가 짧은 것부터

2. 길이가 같으면 사전 순으로

- 첫째 줄에 단어의 개수 N이 주어진다. (1 ≤ N ≤ 20,000) 둘째 줄부터 N개의 줄에 걸쳐 알파벳 소문자로 이루어진 단어가 한 줄에 하나씩 주어진다. 주어지는 문자열의 길이는 50을 넘지 않는다.

- 조건에 따라 정렬하여 단어들을 출력한다. 단, 같은 단어가 여러 번 입력된 경우에는 한 번씩만 출력한다.

```
package com.example;

import java.util.Arrays;
import java.util.Comparator;
import java.util.Scanner;

public class 단어정렬 {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        int n = scanner.nextInt();
        String[] arr = new String[n];

        scanner.nextLine();

        for(int i = 0; i < n; i++) {
            arr[i] = scanner.nextLine();
        }

        Arrays.sort(arr, new Comparator<String>() {
            @Override
            public int compare(String o1, String o2) {
                if(o1.length() == o2.length()) {
                    return o1.compareTo(o2);
                } else {
                    return o1.length() - o2.length();
                }
            }
        });

        System.out.println(arr[0]);
        for (int i = 1; i < n; i++) {
            if (!arr[i].equals(arr[i - 1])) {
                System.out.println(arr[i]);
            }
        }

    }
}

```

1. 입력받은 문자열들을 배열에 저장시키며 Arrays의 sort함수로 조건에 맞게 정렬

2. 두 문자열의 길이가 같으면 사전순으로 정렬하기 위하여 compareTo 함수로 정렬 -> compareTo()함수는 호출하는 객체가 인자보다 사전적으로 순서가 앞설 때 양수를 리턴 

3. 중복된 문자들은 또 출력하지 않기 위하여 0인덱스에 저장된 문자를 출력 후 1부터 반복문 순회하면서 출력

**  N개의 단어를 입력하고 난 후 해당 단어들을 입력받기 전에 개행문자(엔터)를 제거하기위해 추가해야 한다.. 
- 개행문자는 다음에 호출된 Scanner.nextLine( ) 메소드의 입력으로 처리되서 문제가 발생함 // 이 문제로 꽤 시간 많이 잡아먹었다.
