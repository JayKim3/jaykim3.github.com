> 거스름돈

- 타로는 자주 JOI잡화점에서 물건을 산다. JOI잡화점에는 잔돈으로 500엔, 100엔, 50엔, 10엔, 5엔, 1엔이 충분히 있고, 언제나 거스름돈 개수가 가장 적게 잔돈을 준다. 타로가 JOI잡화점에서 물건을 사고 카운터에서 1000엔 지폐를 한장 냈을 때, 받을 잔돈에 포함된 잔돈의 개수를 구하는 프로그램을 작성하시오.

```
package com.example;

import java.io.BufferedReader;
import java.io.IOException;
import java.io.InputStreamReader;

public class 거스름돈 {
    public static void main(String[] args) throws IOException {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));

        int yen = 1000;
        int count = 0;
        int[] coinArr = { 500, 100, 50, 10 , 5, 1 };

        int money = Integer.parseInt(br.readLine());

        money = yen - money;

        for(int i=0; i<coinArr.length; i++) {
            if(money / coinArr[i] > 0) {
                count += money / coinArr[i];
                money = money % coinArr[i];
            }
        }

        System.out.println(count);

    }
}

```

1. 최소한의 거스름돈으로 나눠줄 수 있도록 동전 금액을 큰 수부터 coinArr 배열에 담는다.

2. 500엔 ~ 1엔 까지 나누는 작업을 하면서 count 변수에는 나누어지는 개수를 계속해서 더해주며 money 변수에는 1엔으로 나누어질때까지 나머지금액을 담는다.
